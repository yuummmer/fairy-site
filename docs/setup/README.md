# Setup Documentation

This directory contains setup guides for various integrations and tools used in the FAIRy website.

## Contents

- **GOOGLE_SHEETS_SETUP.md** - Guide for setting up Google Sheets integration for form submissions
- **GOOGLE_SLIDES_SETUP.md** - Guide for creating Google Slides templates that match FAIRy branding
- **INSTITUTION_SURVEY_SETUP.md** - Guide for setting up and managing the institution survey
- **SURVEY_DESIGN.md** - Design and strategy documentation for the 60-second survey
