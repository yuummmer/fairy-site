### Custom Color Palette
Go to **Theme** ‚Üí **Colors** ‚Üí **Custom** and add these exact colors:

**Primary Colors:**
- **Purple Primary**: `#7C3AED` (main buttons, accents)
- **Purple Dark**: `#4C1D95` (headings, main text)
- **Purple Light**: `#6B46C1` (secondary text)

- **Light Purple 2**: `#E9D5FF` (borders, gradients)
- **White**: `#FFFFFF` (cards, content areas)

### Apply Theme Colors
1. **Background**: Use `#FAF5FF` for slide backgrounds
2. **Text**: Use `#4C1D95` for headings, `#6B46C1` for body text
3. **Accents**: Use `#7C3AED` for buttons and highlights

## üî§ Step 3: Set Up Typography

### Recommended Fonts (closest to your website)
1. **Primary**: Roboto (Google's system font)
2. **Secondary**: Open Sans
3. **Fallback**: Arial

### Font Sizes
- **Slide Title**: 36-44pt
- **Section Headers**: 24-28pt  
- **Body Text**: 16-18pt
- **Captions**: 12-14pt
