# Documentation

This directory contains documentation for the FAIRy website project. Documentation is organized by purpose and visibility.

## Structure

### Public documentation

These files are intended to be shared and are at the root of the `docs/` directory:

- **ARCHITECTURE.md** - Technical architecture documentation for developers
- **BRANDING.md** - Brand guidelines, email signatures, and visual identity

### Private documentation

**`internal/`** - Private/internal documentation (git-ignored, not shared publicly)

- **ANALYTICS.md** - Guide for viewing and understanding site analytics
- **ARCH_NOTES_PRIVATE.md** - Private architecture notes and decisions

### Setup guides

**`setup/`** - Configuration and setup documentation

- **GOOGLE_SHEETS_SETUP.md** - Google Sheets integration setup
- **GOOGLE_SLIDES_SETUP.md** - Google Slides template setup
- **INSTITUTION_SURVEY_SETUP.md** - Institution survey setup guide
- **SURVEY_DESIGN.md** - Survey design and strategy
- **README.md** - Overview of setup documentation

### Strategy documentation

**`strategy/`** - Strategic planning and content documentation

- **DEMO_REQUEST_STRATEGY.md** - Demo request feature strategy
- **institution-kit-content.md** - Institution kit content outline
- **README.md** - Overview of strategy documentation

### Presentation materials

**`slides/`** - Slide deck templates and materials

- **SLIDE_DECK_README.md** - Guide for creating slide decks
- **GOOGLE_SLIDES_TEMPLATE_GUIDE.md** - Google Slides template guide
- **slide-deck-template.html** - HTML template for slides
- **fairy-slide-deck.pdf** - Example slide deck

## Quick reference

- **Need technical details?** → `ARCHITECTURE.md`
- **Need brand assets?** → `BRANDING.md`
- **Setting up integrations?** → `setup/`
- **Planning features?** → `strategy/`
- **Creating presentations?** → `slides/`
- **Internal notes?** → `internal/` (private)

## Notes

- Files in `internal/` are git-ignored and will not be committed to the repository
- Public documentation (root level) is intended for sharing with contributors and stakeholders
- Setup and strategy docs may contain sensitive information and should be reviewed before sharing externally
